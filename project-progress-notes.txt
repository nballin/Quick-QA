QuickQA вАФ progress so far (simple notes)
========================================

What works end-to-end (default stack)
-------------------------------------
вАҐ CLI: ingest вЖТ build index вЖТ query вЖТ optional SQuAD eval (`main.py`, `config.yaml`).
вАҐ Encoder: SentenceTransformers MiniLM (`all-MiniLM-L6-v2`) вАФ swap via `encoder.backend` in config.
вАҐ Corpus: `.txt` / `.pdf` / `.docx` under `corpus/`; if empty, falls back to SQuAD passages for ingest.
вАҐ Chunking: size/overlap/sentence split (`pipeline/chunker.py`).
вАҐ Retrieval: FAISS dense + BM25 hybrid with RRF fusion (`pipeline/retrieval.py`).
вАҐ Re-ranking: cross-encoder style scoring on top candidates (`pipeline/scorer.py`).
вАҐ Reading: RoBERTa SQuAD2 extractive answers (`pipeline/reader.py`).
вАҐ Evaluation: EM, F1, recall@k on SQuAD validation (`evaluate.py`).

Phase A / TRM text path (research code вАФ not the default CLI encoder)
---------------------------------------------------------------------
вАҐ `trm_text/`: load ARC TRM checkpoint, UTF-8 byte grids, CRC32вЖТpuzzle-id conditioning,
  vocab bridge (deterministic or learned hook), forward hook on `lm_head`, mean-pool вЖТ embeddings.
вАҐ Documented in `Phase A flowchart.txt`; CNN baseline stub in `trm_text/grid_encoder.py`.
вАҐ `pipeline/encoder.py`: `TRMEncoder` and `CNNEncoder` are placeholders вАФ not connected to
  `TrmArcTextEncoder` yet; using `backend: "trm"` would still raise NotImplementedError.
